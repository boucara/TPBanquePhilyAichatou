Bienvenue sur le gestionnaire de comptes bancaires de Phily et Aichatou.

Fonctionnalité du projet

  • Créer un compte ;
  • Affichage liste des comptes ;
  • Effectuer un transfert d'un compte à un autre ;
  • Effectuer un retrait ;
  • Effectuer un transfert ;
  • Supprimer les comptes ;
  • Afficher la liste des opérations bancaires.
À noter qu'il y a deux façons d'effectuer un retrait ou un dépôt. Soit par le menu adéquate en haut, soit via la liste des comptes, par des popup.

Composants PrimeFaces utilisés mais non étudiés en TP

  • PopUp (http://www.primefaces.org/showcase-labs/ui/dialog.jsf) ;
  • Affichage des messages (http://www.primefaces.org/showcase-labs/ui/messages.jsf) ;
  • Classement dans l'ordre croissant et décroissant des données fonctionnel (ajout d'une collection de comptes bancaires, pour ne pas avoir à la recharger à chaque fois) ;
  • Fonction imprimer pour les opérations bancaires (http://www.primefaces.org/showcase-labs/ui/printer.jsf) ;
  • Utilisation d'icônes (http://www.primefaces.org/showcase-labs/ui/commandButton.jsf).

Autres améliorations

  • Remise en forme des messages d'erreur (centré sur la page) ;
  • Affichage dans la liste des comptes pour les opérations (pas besoin de rentrer les informations).

Points forts

  • Interface esthétique et simple d'utilisation ;
  • Tri fonctionnel ;
  • Utilisation d'un converter pour le choix du compte lors d'un transfert, retrait ou dépôt ;
  • Possibilité d'imprimer la liste des opérations bancaires ;
  • Affichage de message indiquant l'action réalisée.
  • Utilisation d'un weblistener pour générer le jeu de données.

Points faibles

  • Pas de fonction imprimer pour la liste des comptes (problème pour ne pas afficher les 4 dernières colonnes) ;
  • Pas de gestion d'accès (login) ;
  • Possibilité de rentrer des chiffres négatifs pour le montant des opérations.
